# How to Submit (What to Send to the Organization)

Provide the following to Wiz / the hiring panel:
1. **Repository link** (GitHub/GitLab)
2. **Terraform deployment steps** (README section 2)
3. **Demo guide** (`docs/demo-guide.md`)
4. **Architecture overview** (`docs/architecture.md`)
5. **Risk and remediation write-up** (`docs/risks-and-remediations.md`)
6. **Screenshots** (store in a folder in the repo or keep locally):
   - Terraform apply outputs
   - EKS nodes, Ingress hostname
   - `cat /wizexercise.txt` from running container
   - Mongo query results from inside cluster
   - Public S3 listing of backups
   - CloudTrail + GuardDuty enabled
   - Wiz findings view (if applicable)

If the recruiter provided a slide template, copy the key sections above into that template.
